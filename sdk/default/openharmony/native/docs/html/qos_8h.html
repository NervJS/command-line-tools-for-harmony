Detailed Description

Declares the QoS interfaces in C.

Quality-of-service (QoS) refers to the priority scheduling attribute of tasks in OpenHarmony. Developers can use QoS to categorize tasks to be executed to indicate the degree of their relevance to user interactions, the system can schedule the time and running order of tasks according to the QoS set by the tasks.
