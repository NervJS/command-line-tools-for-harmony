Detailed Description

Provides Base DDK APIs, including creating the shared memory, mapping the shared memory,
unmapping the shared memory, and destroying the shared memory.

Provides Base DDK types and declares the macros, enums, and
data structs used by the Base DDK APIs.

DDK_RetCode OH_DDK_CreateAshmem ( const uint8_t *  name,
uint32_t  size,
DDK_Ashmem **  ashmem 
)

Creates shared memory. To prevent resource leakage, destroy the shared memory that is not required by
calling OH_DDK_DestroyAshmem.

Parameters
namePointer to the shared memory to create.
sizeSize of the buffer corresponding to the shared memory.
ashmemPointer to the shared memory created.
Returns
DDK_SUCCESS the operation is successful. DDK_INVALID_PARAMETER name is NULL, size is 0 or ashmem is NULL. DDK_FAILURE create the shared memory failed or create structure DDK_Ashmem failed.
