◆ OH_AI_TensorSetUserData()

OH_AI_API OH_AI_Status OH_AI_TensorSetUserData ( OH_AI_TensorHandle  tensor,
void *  data,
size_t  data_size 
)

Set the data for the tensor with user-allocated data buffer.

The main purpose of this interface is providing a way of using memory already allocated by user as the Model's input, but not which allocated inside the Model object. It can reduce one copy.
Note: The tensor won't free the data provided by invoker. Invoker has the responsibility to free it. And this free action should not be preformed before destruction of the tensor.

Parameters
tensorTensor object handle.
dataA pointer to the user data buffer.
datathe byte size of the user data buffer.
Returns
OH_AI_STATUS_SUCCESS if success, or detail error code if failed.
