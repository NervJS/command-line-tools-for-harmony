Wifi_ResultCode OH_Wifi_IsWifiEnabled ( bool *  enabled)

Check whether the wifi switch is enabled.

Parameters
enabled- It is a boolean pointer used to receive wifi switch status values.
Equal to true indicates that the wifi switch is turned on, false indicates that
the wifi switch is turned off.
The caller needs to pass in a non empty boolean pointer, otherwise an error will be returned.
Returns
wifi functions result code.
For a detailed definition, please refer to Wifi_ResultCode.
WIFI_SUCCESS Successfully obtained the wifi switch status.
WIFI_INVALID_PARAM The input parameter enabled is a null pointer.
WIFI_OPERATION_FAILED Internal execution failed.
