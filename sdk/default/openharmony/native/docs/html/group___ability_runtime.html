Provide the definition of the C interface for the native AbilityRuntime.

Describe the functions provided by the start options.

Describe the constant of context.

Describe the functions provided by the application context.

Since

Enumerator
ABILITY_RUNTIME_AREA_MODE_EL1 

System level device encryption area.

ABILITY_RUNTIME_AREA_MODE_EL2 

User credential encryption area.

ABILITY_RUNTIME_AREA_MODE_EL3 

User credential encryption area. when screen locked, can read/write, and create file.

ABILITY_RUNTIME_AREA_MODE_EL4 

User credential encryption area. when screen locked, FEB2.0 can read/write, FEB3.0 can't read/write, and all can't create file.

ABILITY_RUNTIME_AREA_MODE_EL5 

User privacy-sensitive encryption area. when the screen locked, a closed file cannot be opened, read, or written, a file can be created and then opened, read, or written.
