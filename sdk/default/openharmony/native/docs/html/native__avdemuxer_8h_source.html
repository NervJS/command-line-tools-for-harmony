OH_AVDemuxer_SelectTrackByID
OH_AVErrCode OH_AVDemuxer_SelectTrackByID(OH_AVDemuxer *demuxer, uint32_t trackIndex)
The specified track is selected and the demuxer will read samples from this track. Multiple tracks are selected by calling this interface multiple times with different track indexes. Only the selected tracks are valid when calling OH_AVDemuxer_ReadSample to read samples. The interface returns AV_ERR_OK and the track is selected only once if the same track is selected multiple times. SystemCapability.Multimedia.Media.Spliter.
