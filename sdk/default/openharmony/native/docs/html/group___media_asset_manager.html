Detailed Description

Provides APIs of request capability for Media Source.

Since
12

The OH_MediaAssetManager structure and MediaLibrary_RequestId type are used to request media library resources. The request can be cancelled using the request ID.

Since
12
13

typedef struct MediaLibrary_RequestId MediaLibrary_RequestId

Define MediaLibrary_RequestId.

This type is returned when requesting a media library resource. The request id is used to cancel the request. The value is all zero like "00000000-0000-0000-0000-000000000000" if the request fails.
