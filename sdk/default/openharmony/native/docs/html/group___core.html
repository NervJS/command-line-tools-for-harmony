◆ OH_AVCodecBufferFlags

enum OH_AVCodecBufferFlags

Enumerate the categories of OH_AVCodec's Buffer tags. SystemCapability.Multimedia.Media.Core.

Since
9
Enumerator
AVCODEC_BUFFER_FLAGS_EOS 

Indicates that the Buffer is an End-of-Stream frame.

AVCODEC_BUFFER_FLAGS_SYNC_FRAME 

Indicates that the Buffer contains keyframes.

AVCODEC_BUFFER_FLAGS_INCOMPLETE_FRAME 

Indicates that the data contained in the Buffer is only part of a frame.

AVCODEC_BUFFER_FLAGS_CODEC_DATA 

Indicates that the Buffer contains Codec-Specific-Data.

AVCODEC_BUFFER_FLAGS_DISCARD 

Flag is used to discard packets which are required to maintain valid decoder state but are not required for output and should be dropped after decoding.

Since
12
AVCODEC_BUFFER_FLAGS_DISPOSABLE 

Flag is used to indicate packets that contain frames that can be discarded by the decoder, I.e. Non-reference frames.
