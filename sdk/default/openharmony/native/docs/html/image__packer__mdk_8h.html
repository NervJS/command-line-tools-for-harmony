Detailed Description

Declares APIs for encoding image into data or file.

The packing image data module used to pack pixel data into a target.

The following steps are recommended for packing process: Create a image packer object by calling OH_ImagePacker_Create function. And then covert the image packer object to ImagePacker_Native by OH_ImagePacker_InitNative. Next using OH_ImagePacker_PackToData or OH_ImagePacker_PackToFile to pack source to target area with requird packing options. Finally, release the ImagePacker_Native by OH_ImagePacker_Release.
