◆ napi_event_mode

enum napi_event_mode

Indicates the running mode of the native event loop in an asynchronous native thread.

Since
12
Enumerator
napi_event_mode_default 

In this mode, the current asynchronous thread will be blocked and events of native event loop will be processed.

napi_event_mode_nowait 

In this mode, the current asynchronous thread will not be blocked. If there are events in the event loop, only one event will be processed and then the event loop will stop. If there are no events in the loop, the event loop will stop immediately.

Definition at line 53 of file common.h.

◆ napi_task_priority

enum napi_task_priority

Indicates the priority of a task dispatched from native thread to ArkTS thread.

Since
12
Enumerator
napi_priority_immediate 

The immediate priority tasks should be promptly processed whenever feasible.

napi_priority_high 

The high priority tasks, as sorted by their handle time, should be prioritized over tasks with low priority.

napi_priority_low 

The low priority tasks, as sorted by their handle time, should be processed before idle priority tasks.

napi_priority_idle 

The idle priority tasks should be processed immediately only if there are no other priority tasks.
