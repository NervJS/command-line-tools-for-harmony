Detailed Description

Provides native functions for the resource manager to operate raw file directories and their raw files.

You can use the resource manager to traverse, open, seek, read, and close raw files.

Since
8
Version
1.0

◆ OH_ResourceManager_SeekRawFile()

int OH_ResourceManager_SeekRawFile ( const RawFile *  rawFile,
long  offset,
int  whence 
)

Uses the 32-bit data type to seek a data read position based on the specified offset within a raw file.

Parameters
rawFileIndicates the pointer to RawFile.
offsetIndicates the specified offset.
whenceIndicates the new read position, which can be one of the following values:
0: The new read position is set to offset.
1: The read position is set to the current position plus offset.
2: The read position is set to the end of file (EOF) plus offset.
Returns
Returns (int) 0 if the operation is successful; returns (int) -1 if an error occurs.
