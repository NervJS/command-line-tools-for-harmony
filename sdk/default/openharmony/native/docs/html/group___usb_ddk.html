Detailed Description

Provides USB DDK APIs to open and close USB interfaces, perform non-isochronous and isochronous
data transfer over USB pipes, and implement control transfer and interrupt transfer, etc.

Provides USB DDK types and declares the macros, enumerated variables, and
data structures required by the USB DDK APIs.
