Detailed Description

Provides debug functions.

Provides debug code define.

For example, you can use these functions to obtain cpu uage, memory, heap, capture trace.

For example, you can use these code for check result or parameter of HiDebug function.

)

Start capture application trace.

Parameters
flagTrace flag
tagsTag of trace
limitSizeMax size of trace file, in bytes, the max is 500MB.
fileNameOutput trace file name buffer
lengthOutput trace file name buffer length
Returns
0 - Success HIDEBUG_INVALID_ARGUMENT 401 - if the fileName is null or the length is too short or limitSize is too small 11400102 - Have already capture trace 11400103 - Have no permission to trace 11400104 - The Possible reason is some error in the system.
