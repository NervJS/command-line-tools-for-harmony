Detailed Description

Vulkan APIs for the GPU spatial upscaling of XEngine. Before using APIs in this header file, you need to call the HMS_XEG_EnumerateDeviceExtensionProperties API to verify that extension XEG_SPATIAL_UPSCALE_EXTENSION_NAME is available. SystemCapability.Graphic.XEngine.
