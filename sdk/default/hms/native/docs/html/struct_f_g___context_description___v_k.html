◆ framesInFlight

uint8_t FG_ContextDescription_VK::framesInFlight

Specifies how many frames can be "in flight". For example, if the next frame is being rendered ONLY AFTER the previous one has been presented, than framesInFlight should be set to 1. For example, if the next frame is being rendered WHILE the previous one is being presented, than framesInFlight should be set to 2. Note: Must not be zero!
