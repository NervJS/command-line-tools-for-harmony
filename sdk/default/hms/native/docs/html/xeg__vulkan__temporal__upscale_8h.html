Detailed Description

AI temporal upscaling APIs of XEngine. The recommended upscaling multiple range is [1.25, 2.0]. Before using APIs in this header file, you need to call the HMS_XEG_EnumerateDeviceExtensionProperties API to verify that extension XEG_TEMPORAL_UPSCALE_EXTENSION_NAME is available.
