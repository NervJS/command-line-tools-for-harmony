Detailed Description

Vulkan APIs for the adaptive VRS of XEngine. Before using APIs in this header file, you need to call the HMS_XEG_EnumerateDeviceExtensionProperties API to verify that extension XEG_ADAPTIVE_VRS_EXTENSION_NAME is available. SystemCapability.Graphic.XEngine.
