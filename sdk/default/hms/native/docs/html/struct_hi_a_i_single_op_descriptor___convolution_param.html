HiAI_SingleOpPadMode HiAISingleOpDescriptor_ConvolutionParam::padMode

Input padding mode. For HIAI_SINGLEOP_CONV_MODE_COMMON and HIAI_SINGLEOP_CONV_MODE_DEPTHWISE, the value can be 0 (SPECIFIC), 1 (SAME), 2 (SAME_UPPER), 3 (SAME_LOWER), or 4 (VALID). For HIAI_SINGLEOP_CONV_MODE_TRANSPOSED, the value can be 0 (SPECIFIC), 1 (SAME), or 4 (VALID).
