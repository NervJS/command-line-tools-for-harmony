Detailed Description

Defines the graphic-platform-insensitive APIs of OpenGTX.

Allows you to create OpenGTX context, configure context attributes, activate context, deactivate context, dipatch frameRenderInfo, dispatch gameSceneInfo, dispatch networkinfo and destroy context instance.
