Detailed Description

Defines the APIs of Frame Generation dedicated to the Vulkan API platform.

GraphicsAccelerateKit

Allows you to create Frame Generaion context on Vulkan platform, configure context attributes, activate context, deactivate context, create input and output image instance, draw predicted frames with dispatch Info and destroy context instance.
