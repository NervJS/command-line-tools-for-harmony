Detailed Description

Defines the APIs of Frame Generation dedicated to the OpenGL ES API platform.

GraphicsAccelerateKit

Allows you to create Frame Generaion context on OpenGL ES platform, configure context attributes, activate context, deactivate context, draw predicted frames with dispatch Info and destroy context instance.
